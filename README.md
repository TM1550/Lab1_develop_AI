# Lab1_develop_AI


Установка:

```git clone https://github.com/TM1550/Lab1_develop_AI.git```

```cd Lab1_develop_AI```

```pip install -r requirements.txt```

example.py пример для генерации ответа на вопрос и получения метрик на датасете   

Запуск:

```python example.py```
